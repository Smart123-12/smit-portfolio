## 🚀 Getting Started (Local Development)

```bash
# Clone the repository
git clone https://github.com/Smart123-12/smit-portfolio.git

# Navigate to project
cd smit-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
